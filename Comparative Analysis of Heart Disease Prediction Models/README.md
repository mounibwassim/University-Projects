# Comparative Analysis of Heart Disease Prediction Models

## 🎯 Objective
This computational intelligence project provides a rigorous comparative analysis of various machine learning models to predict the presence of heart disease in patients. The objective is to identify the most reliable algorithmic approach for diagnostic predictions.

## 🛠️ Tools & Technologies
* **Python 3.x**
* **scikit-learn**: For implementing and comparing standard machine learning models.
* **Pandas / NumPy**: For data wrangling, preprocessing, and exploratory data analysis.
* **Matplotlib / Seaborn**: For visualizing statistical trends, correlation matrices, and model performance.

## 📊 Dataset
The analysis relies on the **UCI Heart Disease Dataset** (`heart.csv`), which contains crucial medical attributes (e.g., age, cholesterol levels, chest pain type, maximum heart rate) for comprehensive diagnostic modeling.

## 📈 Results / Output
Multiple models are trained and benchmarked against each other. Key performance indicators (KPIs) such as Accuracy, Precision, Recall, and F1-Score are visualized across models to recommend the optimal algorithmic framework for heart disease diagnostics.
