## 🎯 Objective
This project focuses on designing and training a Convolutional Neural Network (CNN) from scratch to classify complex images. The goal is to build an effective model that can accurately discern image features, identifying the correct class among multiple categories. 

## 🛠️ Tools & Technologies
* **Python**: Core programming language.
* **TensorFlow / Keras**: Deep learning framework used for building and training the CNN architecture.
* **NumPy & Matplotlib**: For array manipulations and visualization of training metrics/results.
* **Jupyter Notebook**: Interactive development environment.

## 📊 Dataset
This project generally utilizes established computer vision datasets (like CIFAR-10) to train the model, demonstrating practical competence with RGB image data, tensor conversions, and categorical labeling.

## 📈 Results / Output
The notebook successfully establishes a functional deep learning pipeline, showing accurate classification of unseen testing images. Training accuracy and validation curves are plotted to monitor overfitting and evaluate parameter tuning effectiveness.
