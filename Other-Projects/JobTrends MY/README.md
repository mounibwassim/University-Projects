# Job Trends MY

An analysis and interactive dashboard of job trends and unemployment statistics in Malaysia.

## Overview
This project contains an interactive UI/UX dashboard built to explore and visualize Labor Force Survey (LFS) data across various states in Malaysia. It includes data preprocessing, exploratory data analysis, and an HTML dashboard output.

## Contents
- **`TDW_UIUX_Dashboard.ipynb`**: The main Jupyter Notebook containing the data analysis and dashboard generation code.
- **`tdw_unemployment_dashboard.html`**: The exported interactive HTML dashboard.
- **CSV Data Files**: Cleaned labor force survey datasets used for the analysis.
