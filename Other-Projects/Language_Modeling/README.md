# Language Modeling with N-Grams

This project implements a statistical Language Model using **N-Grams** (specifically Bigrams and Trigrams) to predict the next word in a sequence and generate coherent text.

## 🚀 Overview
Language modeling is a fundamental task in Natural Language Processing (NLP) that involves predicting the probability of a sequence of words. This notebook demonstrates:
- **Text Preprocessing**: Tokenization and normalization of raw text data.
- **N-Gram Construction**: Building Bigram and Trigram models from the corpus.
- **Probability Estimation**: Calculating conditional probabilities for word sequences.
- **Text Generation**: Iteratively predicting the next word to generate random, yet contextually relevant, sentences.

## 🛠️ Key Features
- **N-Gram Model**: Implementation of n-gram statistics to capture local linguistic patterns.
- **Probability Distribution**: Exploring the likelihood of various word continuations.
- **Random Text Generator**: A script to generate synthetic text based on the trained model.

## 📈 Results
The model demonstrates how simple statistical methods can capture the structure of a language and be used for basic generative tasks.
