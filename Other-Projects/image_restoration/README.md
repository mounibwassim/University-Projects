## 🎯 Objective
This computer vision project is engineered to perform advanced image restoration. The core objective is to recover or enhance corrupted, noisy, or degraded images back to their original or highly-approximated pristine states using computational techniques.

## 🛠️ Tools & Technologies
* **Python 3.x**
* **OpenCV / scikit-image**: Dominant libraries utilized for powerful image processing array manipulations and filtering.
* **NumPy**: Fundamental for operating on image pixel matrices.
* **Matplotlib**: For Before & After comparative visual analysis.

## 📊 Focus Areas
The notebook specifically targets common image degradation problems, such as noise removal (denoising), deblurring, and contrast adjustments across both grayscale and RGB visuals. 

## 📈 Results / Output
The notebook provides working functions that successfully output restored images, retaining crucial edge definitions and structural integrity while mitigating unwanted noise or distortion artifacts.
