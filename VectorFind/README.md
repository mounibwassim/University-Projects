# VectorFind

Welcome to **VectorFind**. This project explores Topological Data Analysis (TDA) using Python.

## 📂 Contents
* `TDA code.ipynb`: Jupyter notebook containing the code, visualizations, and analysis for the TDA implementation.

## 🚀 Getting Started
To view or run the code, you can open the Jupyter Notebook directly in GitHub, or use Google Colab for an interactive environment.

## 🛠️ Technologies Used
* Python
* Topological Data Analysis (TDA) tools (e.g., giotto-tda, ripser)
* Jupyter Notebook
